# Vision-Food-Classifier: A Full-Stack ML Application 🍕🚫

A deep learning-based image classification system designed to distinguish between food and non-food items. This project serves as a practical implementation of the full ML lifecycle, from raw data collection to web deployment.

## 🚀 Project Overview
This project explores the integration of Computer Science fundamentals with Machine Learning. It uses a custom-built dataset and a deep learning architecture to provide real-time inference via a web interface.

### 🛠 Tech Stack
- **Languages:** Python (ML Logic), C++ (Pre-processing utilities), SQL (Metadata management)
- **Frameworks:** Flask, TensorFlow/Keras
- **Libraries:** NumPy, Pandas, Matplotlib, Seaborn
- **Development:** Git, GitHub, VS Code

## 🧠 Key Features & Learning Goals
- **Data Engineering:** Scraping and cleaning 50,000+ images from ImageNet and Food-101.
- **Model Architecture:** Implementing EfficientNet-based Transfer Learning.
- **Mathematics:** Practical application of Binary Cross-Entropy loss and Sigmoid activation functions.
- **SDE Fundamentals:** Managing project structure, environment variables, and Git workflows.
